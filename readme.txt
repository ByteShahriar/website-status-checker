== Description ==

The Website Status Checker plugin allows you to check the status of websites and pages. It provides two shortcodes that enable you to display a status check form and the corresponding status result.

Use the following shortcodes:
- `[website_status_form]` to display the status check form.
- `[check_status_result]` to display the status result.

== Installation ==

1. Upload the 'website-status-checker' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin using the instructions provided in the WordPress dashboard.

== Frequently Asked Questions ==

= How do I use the shortcodes? =

To check the status of a website or page, you can use the following shortcodes:
- `[website_status_form]` - Display the status check form.
- `[check_status_result]` - Display the status result.

= Where can I find the plugin settings? =

You can find the plugin settings and instructions in the WordPress dashboard under the "Status Checker" menu.
